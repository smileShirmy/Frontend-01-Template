## 有限状态机处理字符串

### 有限状态机

1. 每个状态都是一个机器在每哥机器里，我们可以做计算，存储，输出；所有机器接收的
   输入是一致的；状态机的每一个机器本身没有状态，如果我们用函数来表示的话，他应
   该是纯函数（无副作用）
2. 每个机器知道下一个状态每个机器都有确定的下一个状态（moore）；每个机器根据输入
   决定下一个状态（mealy）；

### 浏览器的实现

1. 收集 CSS 规则;
2. 添加调用， 创建 element 的时候获取 CSS 计算；
3. 获取父元素序列；
4. 拆分选择器；
5. 计算选择器与元素匹配；
6. 生成 computed 属性；
7. 确定规则覆盖关系；

### 方法

STAR

1. 情境(situation);
2. 目标（target）;
3. 行动(action);
4. 结果(result);
